AI를 활용한 마케팅의 변화

F&CO 인플루언서 대시보드 v2.0
트렌드를 반영하고 광고 맞춤형 소재로 확장

기존 '콘텐츠 관리'에 머물렀던 대시보드를 외부 소셜 데이터를 AI가 분석하여 반응률이 좋은 트렌드를 적용하고, 변경된 광고 로직에 대응하는 '소재 중심의 캠페인 운영 플랫폼'으로 도약합니다.

모듈 살펴보기 파이프라인 5단계 보기
VALUE 01
트렌드 컨텐츠 크롤링

브랜드 콘텐츠 크롤링·분석·평가까지 반복 업무를 자동화해 실무진의 시간을 확보합니다.

VALUE 02
AI 캠페인 기획

제품 정의부터 전략·콘텐츠 제작·인플루언서 매칭까지 AI가 초안을 만들고 실무자가 보완합니다.

VALUE 03
인플루언서 데이터 자산화

국가별 콜라보 이력과 자사 시딩 인플루언서를 한 곳에 축적해 다음 캠페인의 출발선으로 삼습니다.

MODULE 02 / 캠페인 빌더

Phase 1~8로 캠페인 한 건을 분해합니다

AI 보조와 실무자 의사결정을 8단계로 나눠, 각 Phase가 다음 Phase의 입력이 되도록 설계되어 있습니다.

01
PDA

제품 정의와 캠페인 컨셉을 AI가 자동 생성하고 실무진이 다듬습니다.

02
전략

메시징·채널·내러티브 아크·정렬 점수로 캠페인 전략을 구조화합니다.

03
콘텐츠 로드맵

캠페인 기간 동안의 콘텐츠 아크를 시각화해 운영 일정을 한눈에 봅니다.

04
콘텐츠 제작

5스텝 워크플로우로 AI 이미지·영상을 9:16 세로 비율로 생성합니다.

05
인플루언서 선정

AI 매칭과 페르소나 그룹핑으로 캠페인에 맞는 인플루언서를 선정합니다.

06
인플루언서 소통

선정된 인플루언서에게 브리프를 전달하고 참여 의사를 확정합니다.

07
콘텐츠 론칭

제작된 콘텐츠를 검수·승인하고 일정에 맞춰 론칭합니다.

08
모니터링

Campaign Hub KPI 영역에서 캠페인 성과를 실시간으로 추적합니다.

MODULE 03 / 인플루언서 풀

매번 새로 찾지 않고 풀에서 출발합니다

외부 발굴 툴과 연동해 후보를 확보하고, 콜라보·시딩 이력을 자산으로 축적합니다.

POOL 01
인플루언서 찾기 툴 연동

외부 인플루언서 발굴 툴과 연결해 후보 인플루언서 데이터를 대시보드 안으로 가져옵니다.

POOL 02
나라별 브랜드 콜라보 풀

국가별로 브랜드와 콜라보한 이력이 있는 인플루언서를 분류해 관리합니다.

POOL 03
자사 시딩 인플루언서 풀

자사 시딩이 진행됐거나 완료된 인플루언서를 풀로 축적해 다음 캠페인에 즉시 활용합니다.

MODULE 04 / 인플루언서 컨텐츠 대시보드

크리에이터 콘텐츠를 한 화면에서 추적합니다

시딩된 콘텐츠와 자사 태그 UGC를 인플루언서 단위로 묶어, 성과와 인사이트를 즉시 확인합니다.

VIEW 01
인플루언서별 콘텐츠 뷰

각 인플루언서가 업로드한 콘텐츠를 한곳에 모아 캠페인 진행 현황과 톤을 한눈에 확인합니다.

VIEW 02
콘텐츠 성과 메트릭

조회수·인게이지먼트·별점 지표를 시각화해 우수 콘텐츠와 다음 캠페인 시드 후보를 발굴합니다.

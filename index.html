AI를 활용한 마케팅의 변화

F&CO 인플루언서 대시보드 v2.0
트렌드를 반영하고 광고 맞춤형 소재로 확장

기존 '콘텐츠 관리'에 머물렀던 대시보드를 외부 소셜 데이터를 AI가 분석하여 반응률이 좋은 트렌드를 적용하고, 변경된 광고 로직에 대응하는 '소재 중심의 캠페인 운영 플랫폼'으로 도약합니다.

모듈 살펴보기
VALUE 01
트렌드 컨텐츠 크롤링

브랜드 콘텐츠 크롤링·분석·평가까지 반복 업무를 자동화해 실무진의 시간을 확보합니다.

VALUE 02
AI 캠페인 기획

제품 정의부터 전략·콘텐츠 제작·인플루언서 매칭까지 AI가 초안을 만들고 실무자가 보완합니다.

VALUE 03
인플루언서 데이터 자산화

국가별 콜라보 이력과 자사 시딩 인플루언서를 한 곳에 축적해 다음 캠페인의 출발선으로 삼습니다.

MODULE 01 / 콘텐츠 엔진

UGC를 자동으로 수집하고 가치까지 판별합니다

자사 브랜드 태그 콘텐츠와 바이럴 뷰티 영상을 모아 분석하고 실무진과 공유합니다.

SNS UGC 모니터링

세 단계 흐름으로 SNS(Instagram, TikTok, YouTube)에서 UGC를 자동 수집·분석하고 실무자들끼리 별점으로 가치 판단을 남깁니다.

MODULE 02 / 캠페인 빌더

Phase 1부터 8단계까지 인플루언서 시딩 캠페인을 설계합니다.

AI가 제안하고 실무자 의사결정 진행하는 프로세스로 진행됩니다.

01
PDA기반 캠페인 컨셉 설계

제품 기획안을 기반으로 캠페인 컨셉을 브랜드가 설정한 PDA(Persona / Desire / Awareness)에 맞춰 AI가 자동 생성하고 실무진이 다듬습니다.

02
캠페인 전략

메시징·채널·내러티브 아크·정렬 점수로 캠페인 전략을 구조화합니다.

03
콘텐츠 로드맵

캠페인 기간 동안의 콘텐츠 아크를 시각화해 채널별 운영 일정을 한눈에 봅니다.

04
콘텐츠 제작

컨셉을 기반으로 시딩 가이드 기획안과 AI 이미지·영상을 생성합니다.

05
인플루언서 선정

인플루언서의 페르소나를 분석하여 캠페인에 맞는 인플루언서를 매칭합니다.

06
인플루언서 소통

선정된 인플루언서에게 브리프를 전달하고 참여 의사를 확정합니다.

07
콘텐츠 론칭

업로드 된 인플루언서 시딩 컨텐츠를 자동으로 스크롤하고 인게이지먼트를 측정합니다.

08
모니터링

캠페인 성과를 일간/주간/전체 기간 등으로 분석하고 효과가 좋았던 기획안을 추적합니다.

MODULE 03 / 인플루언서 풀

매번 새로 찾지 않고 풀에서 출발합니다

Influencer Find Agent과 연동되어 인플루언서 후보를 확보하고,자사 시딩 이력을 자산으로 축적합니다.

POOL 01
인플루언서 찾기 툴 연동

Influencer Find Agent과 연동되어 자사 및 경쟁사 인플루언서 데이터를 대시보드 안으로 자동으로 가져옵니다.

POOL 02
인플루언서 심층 분석

인플루언서 풀에서 심층 분석을 통해 최근 게시물의 특징과 페르소나를 분석 가능합니다. 분석된 내용을 기반으로 자사 캠페인에 활용 가능합니다.

MODULE 04 / 인플루언서 컨텐츠 대시보드

콘텐츠를 한 화면에서 확인 가능합니다

시딩된 콘텐츠와 자사 태그 UGC, 뷰티 UGC를 한 화면에서 확인하고, 영상과 댓글 분석을 통해 인사이트를 확인합니다.

VIEW 01
인플루언서별 콘텐츠 뷰

인플루언서가 업로드한 시딩 콘텐츠를 한곳에 모아 캠페인 진행 현황과 톤을 한눈에 확인합니다.

VIEW 02
콘텐츠 성과 메트릭

조회수·인게이지먼트를 추적하고 영상&댓글 분석을 통해 인사이트까지 확인 가능합니다. 해당 인사이트는 다음 캠페인으로 확인 가능합니다.
